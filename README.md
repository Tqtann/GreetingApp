# GreetingApp

A simple Windows Forms application built with C# and .NET Framework 4.8.

## Features

* Enter your name and receive a personalized greeting **(Hello, Titan!)**
* Validation to prevent empty input
* Clear button to reset the text field
* Custom application icon
* Fixed-size window for consistent layout and user experience

## Technologies Used

* C#
* Windows Forms
* .NET Framework 4.8
* Visual Studio 2022

## Installation

1. Download the latest release from the Releases section. **(Release v1.0)**
2. Extract the downloaded files if necessary.
3. Run `GreetingApp.exe`.

## Usage

1. Launch the application. **(GreetingApp.exe)**
2. Enter your name in the text box.
3. Click **Say Hello** to display a greeting.
4. Click **Clear** to reset the input field.
